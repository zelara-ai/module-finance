# Zelara Finance Module

Finance management module for Zelara.

## Overview

Helps users organize personal and business finances with donation suggestions to green initiatives.

**Features**:
- Expense logging and categorization
- Monthly spending breakdown
- Budget analysis
- Donation suggestions (portion of excess funds)

**Unlock**: 50 points

## Tech Stack

- TypeScript (React components for mobile/desktop)
- SQLite (transaction storage)
- @zelara packages (shared types, state)

For detailed technical documentation, see [wiki](wikis/).
